 Description   

The Obs Concept setValueBoolean(boolean), does not appear to work when you pass in a boolean because the line 

https://github.com/openmrs/openmrs-core/blob/bbc37d0d0e55f4ce2526c70a1e1c7dd7d7989ce2/api/src/main/java/org/openmrs/Obs.java#L645

during our testing, never sets anything other than "null" and we need to determine what is causing it first and fix it. 

These two tests should be added to the api/src/test/java/org/openmrs/ObsTest.java to test this method:

 

 

/**
* @see Obs#setValueBoolean(Boolean)
*/
@Test
public void setValueAsBoolean_shouldSucceedWhenProvidedABoolean() {
Obs obs = new Obs();
ConceptDatatype cdt = new ConceptDatatype();
cdt.setUuid(ConceptDatatype.BOOLEAN_UUID);
Concept c = new Concept();
c.setDatatype(cdt);
assertTrue(c.getDatatype().isBoolean());
obs.setConcept(c);
obs.setValueBoolean(Boolean.TRUE);
}

 

/**
* @see Obs#setValueBoolean(Boolean)
