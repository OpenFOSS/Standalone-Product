Comment by Gideon Nyamachere [ 2018-03-17 ]

This ticket was referencing an issue with the version used with the MRNgen module which was breaking when used with newer versions of the core app. Since the MRNgen (https://wiki.openmrs.org/display/docs/MRNGen+Module)  was replaced by the IDGen Module (https://addons.openmrs.org/#/show/org.openmrs.module.idgen) which does not have any references to the mrngen.database_version, we recommend that this issue should be closed.
